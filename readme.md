This code is to be used to pull and store U.S. Senate and House voting records into a database for future processing. 

Currently the senate vote list for fata from 1989 onwards (101st Congress, 1st Session).
	votes and votelists saved as xml.
The house votes are pullable for data from 1990 onwards (101st Congress, 2nd Session).
	votes saved as xml and votelists as csv.

The files saved as xml are done so because they are merely being pulled directly form congressional sources and saved locally.
The house votelist is saved as a csv because the list was stored as multiple html pages and csv seemed good enough at the time.
